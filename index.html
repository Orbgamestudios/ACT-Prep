This app can show saved passages, but it cannot generate new ACT-like practice until you add a Gemini API key.

  1. Go to Google AI Studio API keys.
  2. Sign in with Google and choose Create API key.
  3. Copy the key, open Settings here, and paste it into Gemini API key.
